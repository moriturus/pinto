# **Summary**

Add an assignee filter to `list` and `board` so users can focus on PBIs assigned to a particular person.

# **Acceptance Criteria**

- [ ] `pinto list --assignee <name>` returns only PBIs whose assignee matches the requested name.
- [ ] `pinto board --assignee <name>` applies the same filter to every displayed workflow column.
- [ ] The filter composes with existing status, Sprint, label, and search filters and works with JSON output.
- [ ] Omitting the filter preserves the current results and ordering.
