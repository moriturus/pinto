# **Summary**

Provide a read-only `pinto export --json` command that returns one machine-readable snapshot containing the board's PBIs, Sprints, configuration, and Definition of Done.

# **Acceptance Criteria**

- [ ] `pinto export --json` returns PBIs, Sprints, board configuration, and the shared Definition of Done in one JSON document.
- [ ] The export uses the same field meanings and UTC timestamp format as the existing JSON commands.
- [ ] Running the export does not modify `.pinto` data or require a server.
