## Summary

Investigate and fix the failures observed in CI run 29405214703 (commit afdd887). The macOS check fails in `kanban::pty_tests::shell_can_reenter_kanban_without_leaking_lifecycle_state`: after the second Kanban session returns to the shell, Ctrl-D does not terminate the child within the test's three-second wait on macos-latest (macOS 26 arm64). The coverage job runs all tests successfully but `cargo llvm-cov --all-features --workspace --locked --cobertura --output-path coverage.xml --fail-under-lines 95` exits 1 because the text report total is 94.99%; the generated Cobertura artifact reports 16,739/17,542 lines (95.422%), so the gate and artifact metrics must be reconciled.
