# **Summary**

Add a read-only `pinto next` command that identifies the highest-ranked PBIs that are ready to start.

# **Implementation Plan**

## Candidate selection

- Define an unstarted PBI as one in the first configured workflow column, and exclude items in the configured `done_column` even when a board uses a single-column or unusual workflow.
- Require every declared dependency to resolve to an active PBI in the configured `done_column`; missing dependencies remain blocked and are not returned.
- Reuse canonical backlog/hierarchical ordering, apply the hierarchy before the count cap, use an exact optional Sprint filter, and cap the result at the requested positive count.
- Apply the existing effective-points calculation so JSON fields have the same meaning as `list` and `show`.

## CLI and output

- Add a read-only `next` command with `-n/--count`, `-S/--sprint`, and `-j/--json` options.
- Render human output with the compact PBI list format and JSON output with the existing PBI JSON schema. An empty result reports that no actionable PBI exists.
- Read through the existing backend abstraction without acquiring a write lock, saving items, changing timestamps, or committing data.

## Tests and verification

- Add pure service tests for rank order, hierarchical count limiting, dependency readiness, Sprint filtering, missing dependencies, and completed/already-started exclusion.
- Add CLI integration tests for default output, count/Sprint options, JSON output, configured workflow columns, invalid counts, and read-only behavior; add a focused demo and update the CLI documentation.
- Verify the feature through `cargo run`, run the repository quality checks, then apply iterative review before closing this item.

# **Acceptance Criteria**

- [x] `pinto next` returns the highest-ranked unstarted PBI whose dependencies are all in the configured done column.
- [x] `pinto next -n <count>` returns up to the requested number of eligible PBIs in rank order.
- [x] `pinto next --sprint <id>` restricts candidates to the requested Sprint.
- [x] Blocked or already-completed PBIs are not returned, and the command does not modify the board.
