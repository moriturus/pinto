# **Summary**

Add a stale-item filter that finds PBIs whose `updated` timestamp has not changed for a requested duration.

# **Acceptance Criteria**

- [ ] `pinto list --stale <duration>` returns PBIs older than the duration at the time of the query.
- [ ] A documented duration such as `7d` is accepted, and invalid durations produce an actionable user error.
- [ ] The filter composes with the existing status, Sprint, label, search, and output options.
- [ ] The stale query is read-only and does not update item timestamps.
