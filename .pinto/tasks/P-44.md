# **Summary**

The shared CLI test helper forwards the developer's `LC_ALL` and `LANG` to spawned pinto processes, so English output assertions fail under a Japanese locale (for example `usage::cli_options_have_short_forms`). The suite currently passes only in English-locale environments such as CI runners.

# **Acceptance Criteria**

- [ ] The shared helper pins a fixed English locale for spawned pinto processes.
- [ ] The full test suite passes with `LC_ALL=ja_JP.UTF-8` set in the parent environment.
- [ ] i18n tests that override the locale explicitly keep passing.
