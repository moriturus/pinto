# **Summary**

Separate personal keybindings from shared board configuration and document configuration compatibility.

# **Acceptance Criteria**

- [ ] Keybindings are stored at `$XDG_CONFIG_HOME/pinto/config.toml`.
- [ ] Keybindings are not stored in `.pinto/config.toml`, so personal preferences do not become shared board state.
- [ ] Compatibility guidance distinguishes board configuration from Markdown, SQLite, and JSON data formats.
- [ ] Releases that add board configuration keys state whether older binaries can read the updated configuration and provide downgrade guidance when necessary.
