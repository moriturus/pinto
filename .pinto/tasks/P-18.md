# **Summary**

Parse Markdown Acceptance Criteria checkboxes and expose their completion progress without changing the plain-text item format.

# **Acceptance Criteria**

- [ ] PBI details and the Kanban details popup can display Acceptance Criteria progress as completed over total checkboxes.
- [ ] `list --long` and `board --long` can include the same progress as an opt-in column.
- [ ] Moving a PBI to the configured done column emits a non-blocking warning when its Acceptance Criteria contain unchecked boxes.
- [ ] Checkbox counting leaves the stored Markdown body and existing item data format unchanged.
